# O-ETB source code

O-ETB can be built using the makefile in this directory. It creates an executable
named "etb" in the ../BUILD directory.

O-ETB can also be run by compiling and running it in this src directory as follows.
Execute SWI Prolog with the command "swipl" (or whatever name used to install Prolog).
Then load and compile the source with the command "[etb].". Then run it by
entering the command "etb.". This is the way usually used for development and testing.
