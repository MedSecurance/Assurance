# O-ETB User and Internal documentation

## User Manual
This directory contains old, current and in-progress versions of the User Manual in both Word and PDF forms.
The file name gives both the version of O-ETB to which it applies and the version number of the Manual.
Note that the version number of the manual is not incremented for every minor change.
We would like to say that change tracing is always on to indicate all changes since the last vXX version.
However, sometimes all changes are accepted for readability before checking in the file.
Therefore, there may be multiple distinct "versions" of the document with the same file name.
Users should be vigilant for updates to the file when as reported when a Git pull is performed.
