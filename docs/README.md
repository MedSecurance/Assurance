# O-ETB User and Internal documentation

## User Manual
This directory contains old, current and in-progress versions of the User Manual in both Word and PDF forms. Later versions (thus far versions 9, 10 and 11) are given in LaTeX and PDF forms.
In the earlier versions the file name gives both the version of O-ETB to which it applies and the version number of the Manual. In later versions the name gives only the version number of the manual.
Note that the version number of the manual is not incremented for every minor change.
We would like to say that change tracing is always on to indicate all changes since the last vXX version.
However, sometimes all changes in Word documents are accepted for readability before checking in the file.
Therefore, there may be multiple distinct "versions" of the document with the same file name.
Users should be vigilant for updates to the file when as reported when a Git pull is performed.
