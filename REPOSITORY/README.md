# Repositories of O-ETB created persistent objects

OUTLINES - repository of versions of ACO files + workbench state.
The ACO subsystem stores versions of assurance case outlines under development
and state of the ACO workbench.

CASES - repository of assurance cases organized by case identifier.
For each case there is a distinct repository file. These files are not intended
for direct use by a user of O-ETB.

EVIDENCE - repository of evidence records organized by evidence categories.
Evidence records and other auxiliary files are created under directories for each category.
Each evidence record is associated with a unique evidence counter value.
For each evidence record there is a status file that is created by O-ETB and
updated by the agent for the tool or process that creates the evidence item.
