# Build and deployment procedures and results

etb - executable of the O-ETB tool built by the makefile

O-ETB can also be run by compiling and running it in the ../src directory
Execute SWI Prolog with the command "swipl" (or whatever name used to install Prolog).
Then load and compile the source with the command "[etb].". Then run it by
entering the command "etb.".
